# DMUnLoader (Universal Loader & Result Handler)
[![Build Status](https://app.bitrise.io/app/9e391394-db73-473f-998a-2026373de643/status.svg?token=mL8evw6RHiRtfKSiQ82zuw&branch=develop)](https://app.bitrise.io/app/9e391394-db73-473f-998a-2026373de643)

## Overview

This is a library designed to assist developers in conveniently managing and displaying the current state of an application. The library supports three possible states: `Idle`, `Loading Screen`, `Error Screen`, or `Success Screen`. Only one state can be displayed at a time, ensuring a sequential and consistent behavior.
